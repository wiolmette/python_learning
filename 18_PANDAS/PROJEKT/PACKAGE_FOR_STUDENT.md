# Co wysłać kursantce

Najprostsza wersja paczki:

- całe to repo jako `.zip`,
- krótka wiadomość z informacją:
  - od czego zacząć,
  - jaki dataset pobrać,
  - że nie potrzebuje pełnego serwera SQL.

## Sugerowana wiadomość

Cześć, przygotowałem dla Ciebie mini kurs projektowy wokół `pandas`, analizy danych i podstaw pracy z `sqlite`.

Zacznij proszę od:

1. `README.md`
2. `COURSE_PLAN.md`
3. `START_HERE.md`
4. `tasks/01_setup_and_data_understanding.md`

Do projektu nie potrzebujesz pełnego serwera SQL. Wystarczy Python, `pandas`, notebook i lokalna baza `sqlite`.

## Co warto dopowiedzieć ustnie albo w wiadomości

- To nie jest test "czy umiesz wszystko", tylko projekt do nauki.
- Ważniejsze od perfekcyjnego kodu jest umiejętność wyjaśnienia decyzji.
- Jeśli coś jest niejasne, warto zostawiać pytania i notatki w markdown.

## Opcjonalnie

Jeśli chcesz uprościć start jeszcze bardziej, możesz przed spakowaniem:

- dołączyć link do konkretnego datasetu,
- dołączyć małą próbkę danych,
- dopisać własny termin oddawania kolejnych etapów.
