# Folder `reports`

Tu zapisz końcowe wnioski.

## Co może tu trafić

- raport końcowy w markdown,
- eksport notebooka,
- obrazki z wykresami,
- tabela KPI do wklejenia do podsumowania.

## O czym pamiętać

Raport powinien odpowiadać na pytania:

- co zostało przeanalizowane,
- jakie były ograniczenia,
- jakie są główne wnioski,
- jakie rekomendacje wynikają z danych.
