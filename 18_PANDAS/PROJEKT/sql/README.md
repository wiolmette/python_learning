# Folder `sql`

Tu zapisuj zapytania SQL używane na bazie `sqlite`.

## Po co osobny folder

Nawet przy małym projekcie warto oddzielić:

- zapytania SQL,
- kod Pythona,
- raport końcowy.

To ułatwia porównywanie:

- co liczysz w `pandas`,
- co liczysz w SQL,
- które podejście jest dla Ciebie czytelniejsze.

## Pomysł na zawartość

- `01_orders_per_month.sql`
- `02_revenue_by_category.sql`
- `03_repeat_customers.sql`

Nie musisz mieć wielu plików na start, ale dobrze od początku budować taki nawyk.
