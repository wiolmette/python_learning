# Folder `src`

Tu trafia kod, który chcesz używać więcej niż raz.

## Co warto tu przenosić

- funkcje do wczytywania danych,
- funkcje do czyszczenia danych,
- liczenie KPI,
- pomocnicze mapowania i stałe.

## Po co

Notebook jest dobry do nauki i eksploracji, ale łatwo robi się chaotyczny. Folder `src` uczy pierwszego kroku w stronę bardziej uporządkowanej architektury.

## Przykładowe pliki

- `load_data.py`
- `cleaning.py`
- `metrics.py`
- `utils.py`
